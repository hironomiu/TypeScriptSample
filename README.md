# TypeScript Sample

## 環境

```
$ node -v
v12.14.1
$ npm -v
6.13.6
```

[VSCode](https://azure.microsoft.com/ja-jp/products/visual-studio-code/)

## set up

```
$ npm install typescript
$ npm i express @types/express
$ npm i ts-node ts-node-dev
$ npm i body-parser
```

## サーバ

start
```
$ npm run server
```

watch & auto reload
```
$ npm run watch
```

## ビルド

```
$ npm run build
```

## テスト環境

```
$ npm i jest ts-jest supertest @types/jest @types/supertest
```

テストコード

```
./test/app.spec.ts
```
